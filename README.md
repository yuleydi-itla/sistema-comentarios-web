## 🛠️ Características del Proyecto

Para consolidar nuestros conocimientos, el equipo se dividió abarcando todas las áreas necesarias de una aplicación web completa y colaborando con repositorios de Git:

*   **Frontend Elegante (Diseño)**: Construido con **HTML5, CSS y Bootstrap 5**, asegurando que el panel luzca profesional, moderno, con avatares dinámicos y totalmente adaptable a diferentes pantallas. 
*   **Interactividad con JavaScript (AJAX)**: Para evitar las recargas molestas de pantalla al escribir una opinión, aplicamos **Vanilla Javascript** con la moderna API `fetch()`. Esto envía y solicita datos del servidor silenciosamente proporcionando una experiencia veloz e instantánea.
*   **Backend Intermediario (PHP)**: Desarrollamos nuestras propias interfaces (`guardar.php` y `obtener.php`) para enrutar el tráfico de red de forma segura entre la cara principal interactiva y la capa de almacenamiento en lo profundo del servidor.
*   **Seguridad y Datos (SQLite PDO)**: Se utilizaron *Sentencias Preparadas* y orientación a objetos para prevenir ataques de *SQL Injection*. Además, elegimos **SQLite** por defecto, lo que nos brinda una persistencia asombrosa de datos sin dificultar la portabilidad de nube a nube.

## 🚀 Despliegue en Servidor

Arquitectura configurada con directrices directas para subirse con cero configuraciones a plataformas integradas como **CodeSandbox.io**, cumpliendo cabalmente el punto final de publicación exigido por la rúbrica.
